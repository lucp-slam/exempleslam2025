Présentation de Ethan E

Cette page est dédiée à l'éthane, un hydrocarbure saturé.

Formule brute : C2H6

Propriétés physiques :

  • État à température ambiante : gaz
  • Point d'ébullition : -88.6 °C
  • Point de fusion : -183.3 °C
  • Densité : 1.356 g/L
  • Solubilité dans l'eau : très faible

Applications :

  • Utilisé comme combustible dans les chaudières et les moteurs à combustion interne.
  • Précurseur dans la production de produits chimiques tels que l'éthylène.
  • Utilisé dans la fabrication de plastiques, de solvants et d'autres produits chimiques.

Réactions chimiques :

  • Combustion : C2H6 + 7/2 O2 → 2 CO2 + 3 H2O
  • Halogénation : C2H6 + Cl2 → C2H5Cl + HCl
  • Oxydation : C2H6 + O2 → C2H4 + H2O
